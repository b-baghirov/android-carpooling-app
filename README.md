# Android Carpooling App

This is a junior-level Android application developed with Kotlin.
The project demonstrates user authentication, basic chat functionality,
and Firebase integration.

## Technologies
- Kotlin
- Android Studio
- Firebase Authentication
- Firebase Realtime Database

## Features
- User registration and login
- User profile management
- Basic chat system
